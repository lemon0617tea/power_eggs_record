    Announcements and requests from me.

    First, the reason for adding the new category.
    It used to be only general and individual, with night and nightless.
    The problem with this is that there is too much of a luck factor, and the wave to set a record is almost always fixed.
    In order to reduce this factor, I thought of a remedy and increased the number of individual category records, so that it would be easier to be recorded and people would feel more familiar with power eggs.
    Also, some people feel negatively about failing wave 3 in order to achieve a record.
    The overall and individual records do not have to be cleared, but the cleared records will also be displayed on the web and the records will be registered.

    About recording blank spaces in new categories

    If you have any past records, please apply so that we can register them.
    We will notify you via Mention or Tweet when the record you have applied for is ready to be registered.
